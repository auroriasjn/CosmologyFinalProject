# Cosmology Final Project

This project builds upon the Particle-Mesh $N$-body simulation and analysis code provided in the ASTR 6000 course materials ([Part 1: N-Body](https://npadmana.github.io/ASTR6000-SP26/Lectures/020-pm-nbody.html) | [Part 2: Analysis](https://npadmana.github.io/ASTR6000-SP26/Lectures/021-pm-analysis.html)).

### Installation
To set up the project, create a new Conda environment with Python 3.10 and install the required dependencies:
```bash
pip install -r requirements.txt
```
Once the environment is configured, the provided Jupyter notebooks can be executed directly
